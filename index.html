The WebDev Sandbox

Looks like I've got HTML, CSS, and JavaScript down pat terribly quickly, so this page is my first application of the three of them together via the DOM. DOM is the last thing I'll need to know to start some earnest front-end Web development.

If I can make the most of this testing environment, I can move on to more advanced courses, followed by backend development. Maybe even do it professionally.

TIPS TO REMEMBER:

  • My JavaScript console shortcut is Ctrl + shift + J. Alternatively, use Ctrl + shift + I for the full developer console.
  • Keeping the "other sandboxes" links under that bottom-right panel seems cumbersome when clicking back and forth. Future panels of this type should not be animated in this way. However, this one will keep its animation, just as a demonstration. I've also copied its links to the sidebar.
  • I should finish those Khan courses. It's why I made that sidebar at the top right.
  • Find out what it would take to get a Web development job. It seems to come naturally enough, but at my time of writing this, I've just gotten started.
  • Tentative professional path:
    • Learn enough Web development to work on freelance job postings as exercises, until I can complete them fully, with quality, and before their stated deadlines.
    • Actually take on some freelance projects as the hired developer.
    • Put the sites I develop on LinkedIn.
    • Open up my resume to Web development alongside my engineering degree.
  • Visit Kong Development forums to see where they host their stuff.
  • Develop some of the engineering Web tools that should've existed when I was completing my degree.
  • Be free.
